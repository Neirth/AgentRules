# General Engineering

This skill encompasses universal engineering principles and foundational methodologies applicable across all engineering disciplines:

## Capabilities

- **classical-mathematical** - Mathematical modeling, numerical analysis, optimization, and simulation following MIT, Harvard, Oxford, Berkeley curricula
- **formal-verification** - Mathematical proof of code correctness using theorem proving and model checking (C/C++/Rust only)
- **qa-engineering** - Test-Driven Development, test coverage, mocking, chaos engineering, testing culture

## Usage

Each capability is detailed in the `capabilities/` directory. Reference specific capabilities as needed for your task.
