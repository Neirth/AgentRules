# Computer Science

> *"The best code is not the longest code, it is the most efficient code."*

## Purpose

This skill provides comprehensive computer science and software engineering expertise, covering software architecture, design patterns, testing methodologies, machine learning engineering, reverse engineering, and code review practices. Drawing from industry leaders (Netflix, Google, AWS, Microsoft) and academic foundations (MIT, Stanford, Carnegie Mellon), these references guide the development of robust, maintainable, and scalable software systems.

## Software Engineering Philosophy

Software engineering is the disciplined application of engineering principles to the design, development, and maintenance of software systems. Quality is not an afterthought—it must be built in from the start through clean architecture, comprehensive testing, and systematic code review. These references provide the patterns, practices, and principles to build software that is not just functional, but maintainable, scalable, and robust.
